ek9/vimrc
=========

[ek9/vimrc][0] is a personal `.vimrc` configuration published as a pathogen /
vundle plugin. It is inspired by [tpope/vim-sensible][1].

## Install

Install `ek9/vimrc` using Vundle or Pathogen:

```vim
Bundle 'ek9/vimrc'
```
